Web Development

What is Web Development?

Web development, also known as website development, refers to the tasks associated with creating, building, and maintaining websites and web applications that run online on a browser.

Types of Web Development

There are three main types of web development: front-end development, back-end development, and full-stack development.

Front-end Web Development

The user-facing side of a website. Front-end web developers ensure that visitors can easily interact with and navigate through the website.

Back-end Web Development

The server side of a website. In other words, this is the behind-the-scenes work that creates a websiteâ€™s structure. The front end would not exist without the back end.

Full-stack Web Development

This takes into consideration the entirety of a website, including back-end and front-end development.

Common Programming Languages in Web Development

  • HTML (Hypertext Markup Language)
  • CSS (Cascading Style Sheets)
  • JavaScript

HTML Overview

HTML is the standard language used to create and design webpages. It's the most fundamental markup language used for structuring content on the web.

Why HTML is Great for Beginners

  • Simple Syntax: HTML uses tags to define elements, which are easy to understand and use.
  • Immediate Results: You can see the results of your HTML code almost instantly in a web browser.

What is HTML?

HTML is used to structure content on the web. It tells the web browser how to display text, images, and other elements on a webpage.

HTML = Like A, B, C

HTML5 Introduction

HTML5 is a markup language used for structuring and presenting content on the World Wide Web. It is the fifth and current version of the HTML standard. HTML5 includes detailed processing models to encourage more interoperable implementations; it extends, improves, and rationalizes the markup available for documents, and introduces markup and application programming interfaces (APIs) for complex web applications.

History of HTML

Since the early days of the web, there have been many versions of HTML:

  • 1989: WorldWideWeb invented by Tim Berners-Lee.
  • 1991: HTML invented by Tim Berners-Lee.
  • 1993: HTML+ drafted by Dave Raggett.
  • 1995: HTML Working Group defines HTML 2.0.
  • 1997: W3C Recommended HTML 3.2.
  • 1999: W3C Recommended HTML 4.01.
  • 2000: W3C Recommended XHTML 1.0.
  • 2004: WHATWG (Web Hypertext Application Technology Working Group) formed.
  • 2008: HTML5 WHATWG (Web Hypertext Application Technology Working Group) First Public Draft.
  • 2012: HTML5 WHATWG Living Standard.
  • 2014: HTML5 W3C Final Recommendation HTML5.

What is a File Extension?

A file extension is a three or four-letter code that appears at the end of a filename and indicates the type of file it is. For example, .txt stands for text files, .jpg stands for image files, and .html stands for web pages.

What is a Program?

A program is a piece of code or set of instructions that tells a computer how to perform a task. Programs are written by humans and subsequently read by computing devices.
